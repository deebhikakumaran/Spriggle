## 📦 How to Run Locally

Clone the repository

```bash
  git clone https://github.com/deebhikakumaran/Spriggle.git
```

Navigate to the project directory

```bash
  cd Spriggle
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
